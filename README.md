# NumPy Matrix Analysis — Capstone Project

## Overview
This project focuses on analysing body measurement data of adult male and female participants using Python. It applies numerical and statistical techniques to understand patterns, distributions, and relationships within the dataset.

---

## Dataset
The dataset is based on NHANES (National Health and Nutrition Examination Survey) and includes body measurements such as:
- Weight  
- Height  
- Arm length  
- Leg length  
- Arm circumference  
- Hip circumference  
- Waist circumference  

---

## What I Did
- Loaded and processed data using NumPy matrices  
- Cleaned the dataset by handling missing values  
- Visualised data using histograms and boxplots  
- Computed statistical measures (mean, median, standard deviation, variance, skewness, kurtosis)  
- Calculated Body Mass Index (BMI)  
- Standardised data using z-scores  
- Created scatterplot matrix for relationship analysis  
- Performed correlation analysis (Pearson and Spearman)  
- Computed waist-to-height and waist-to-hip ratios  
- Compared distributions between male and female participants  

---

## Tools and Libraries
- Python  
- NumPy  
- Pandas  
- Matplotlib  
- Seaborn  
- SciPy  
- Google Colab  

---

## Key Findings
- Male participants generally have higher average body weight than female participants  
- BMI is strongly related to weight, waist, and hip measurements  
- Waist-based ratios provide better insight into fat distribution than BMI alone  
- Strong positive relationships exist among weight, waist circumference, hip circumference, and BMI
